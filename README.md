# react-webpack-example
This repository is dedicated for a ReactJS example using features like Babel(javascript compiler) and Webpack(module bundler)

# Installation

```zsh
npm install --save-dev webpack \
    standard \
    webpack-dev-server \
    sync-exec \
    react \
    react-dom \
    babel-core \
    babel-loader \
    babel-preset-es2015 \
    babel-preset-stage-0 \
    babel-preset-env \
    babel-preset-react \
    standard-loader \
    react-hot-loader \
    webpack-validator 
```    

# Running

```zsh
npm start
```
